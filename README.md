# pslayers

Image layering with compositing

Uses Imagick and PHP 7+

##

Holds and manages layers of imagick canvases for easy manipulation and
blending.

### Layer Parameters

* Brightness
* Contrast
* Opacity
* Saturation
* Tint
* Hue

### Layer Blend Modes

* Normal
* Dissolve

#### Darken

* Darken
* Multiply
* Colour Burn
* Linear Burn
* Darker Colour

#### Lighten

* Lighten
* Screen
* Colour Dodge
* Linear Dodge (Add)
* Lighter Colour

#### Contrast

* Overlay
* Soft Light
* Hard Light
* Vivid Light
* Linear Light
* Pin Light
* Hard Mix

#### Inversion

* Difference
* Exclusion


#### Cancelleation

* Subtract
* Divide

#### Component

* Hue
* Saturation
* Colour
* Luminosity

## Mask

* Gradiant Mask
* Transparency Mask
